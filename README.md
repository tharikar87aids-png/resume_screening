# 🧠 AI Resume Screening System

## 📌 Overview
An AI-based system that automatically screens and ranks resumes based on a given job description using Machine Learning and NLP techniques.

## 🚀 Features
- Upload resumes (PDF/DOCX)
- Enter job description
- Skill extraction
- TF-IDF vectorization
- Cosine similarity matching
- Resume ranking by match percentage

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn
- NLTK / spaCy
- HTML, CSS, JavaScript

## ⚙️ How It Works
1. Extract resume text  
2. Convert text into TF-IDF vectors  
3. Compare with job description  
4. Rank resumes using cosine similarity  

## 📈 Benefits
- Saves recruitment time  
- Reduces manual effort  
- Improves candidate shortlisting accuracy
